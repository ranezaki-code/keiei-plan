# 経営計画管理システム

ローソン松本新村店 向けの経営計画管理ツール（単一HTMLファイル）。

公開URL：https://ranezaki-code.github.io/keiei-plan/

## 特徴

- 📱 **どの端末からでもアクセス可能**（PC・スマホ・タブレット）
- ☁️ **GitHub Gist による端末間自動同期**（3秒デバウンスで自動アップロード）
- 🔒 **パスワードロック**（セッション単位・3回失敗で1分ロック）
- 💾 **オフライン対応**（localStorage にキャッシュ、オンライン復帰時に自動同期）

## 画面構成

| タブ | 内容 |
|---|---|
| ⓪ 精算書 | 月次精算書の入力・店利益自動算出 |
| ① 月次目標 | 予測値・利益シミュレーション・週別配分 |
| ② 日販・客数 | 日販推移・時間帯別分析 |
| ③ カテゴリ | 商品カテゴリ別計画 |
| ④ 施策計画 | 実施施策・ROI 管理 |
| ⑤ 先週検証 | 振り返り・改善メモ |
| ⑥ QSC | Quality / Service / Cleanliness スコア |
| ⑦ シーズン | 季節商戦の計画・リファレンス |
| ⑧ 発注リスト | 発注アイテム管理 |
| ⑨ その他 | クラウド同期設定・その他計画 |

## 運用

- 金額はすべて **円単位** で入力
- 画面下部のステータス表示で同期状況を確認（🟢同期中 / 🟡Gist未作成 / ⚪ローカル保存のみ）
- 他端末へのセットアップは「🔗 共有リンクをコピー」で 1 クリック共有

## 更新手順

`index.html` を編集後、以下の流れで反映：

```bash
cd ~/keiei-plan
# index.html を更新
git add index.html
git commit -m "v○: 変更内容を簡潔に記述"
git push
```

push 後 1〜2 分で GitHub Pages が自動再ビルド → 公開 URL に反映される。
